# BitmapGrayscale
Lazy bitmap grayscale converter which only supports RGB color format. Have fun! :smiley:

Build: `g++ -Wall -pedantic -std=c++11 main.cpp -o ./BitmapGrayscale`

Usage: `BitmapGrayscale <image_path>`
